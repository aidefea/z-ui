Popup 弹窗组件

示例

代码示例

展示一个可自定义尺寸的弹窗组件。

<template>
  <div>
    <button @click="getcc">显示</button>
    <zpopup
      v-model:visible="showPopup"
      :width="800"
      :height="500"
      :showFullscreenButton="true"
    >
      <h1>123</h1>
    </zpopup>
  </div>
</template>

<script>
import zpopup from "z-zygui";
export default {
  components: {
    zpopup,
  },
  data() {
    return {
      showPopup: false,
    };
  },
  methods: {
    getcc() {
      this.showPopup = true;
    },
  },
};
</script>

参数说明

参数名类型默认值说明

showFullscreenButtonBooleantrue是否展示全屏按钮
