## 功能特点

- **任务管理**：可以创建、编辑、删除任务。
- **多用户支持**：支持团队成员之间的协作。
- **通知提醒**：任务到期时自动提醒。
- **支持多平台**：支持 Web、桌面和移动端。

- **前端**：React, Vue.js, Tailwind CSS
- **后端**：Node.js, Express
- **数据库**：MongoDB
- **构建工具**：Webpack, Vite
- **测试**：Jest, Cypress
